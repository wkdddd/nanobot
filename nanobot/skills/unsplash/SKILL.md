# Unsplash Search

<!-- AI修改后 -->
Use the `search_unsplash_images` tool when the user asks for real photos, stock-style image references, website images, background photos, moodboard images, or visual inspiration from existing photography.
<!-- ###### -->

Do not use this tool when the user asks to generate a new image from scratch. Use image generation for that.

When returning results, keep photographer attribution and Unsplash page links. Do not claim Unsplash photos were generated by the assistant.

Prefer concise searches with English keywords. Use `orientation` when the user asks for a banner, poster, phone wallpaper, avatar, or square image.

Examples:

```text
search_unsplash_images(
  query="minimal office desk",
  count=5,
  orientation="landscape"
)
search_unsplash_images(
  query="beach sunset",
  count=3,
  color="orange"
)

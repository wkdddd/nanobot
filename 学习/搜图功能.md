# Unsplash 搜图功能计划

## 目标
让 agent 在用户需要真实图片素材时，可以通过 Unsplash 搜索图片，并返回图片链接、作者、描述和来源页面。

## 第一版范围
- 支持关键词搜索图片
- 支持返回 1-10 条结果
- 支持 orientation：landscape / portrait / squarish
- 返回图片预览 URL、Unsplash 页面 URL、作者信息
- 不下载图片
- 不保存 artifact
- 不改 WebUI 展示

## 调用链
用户提出找图需求
-> agent 根据 skill 判断要用 search_unsplash_images
-> tool 校验参数和 API key
-> provider 请求 Unsplash API
-> tool 格式化结果
-> agent 回复用户

## 配置设想
providers.unsplash.apiKey
tools.unsplashSearch.enabled
tools.unsplashSearch.defaultCount
tools.unsplashSearch.maxResults

## Tool Schema 设计

工具名：search_unsplash_images

用途：
当用户需要真实照片、背景图、素材图、参考图时，通过 Unsplash 搜索图片。

参数：
- query: string，必填。搜索关键词，例如 "minimal office", "beach sunset"。
- count: integer，可选。返回数量，范围 1-10，默认 5。
- orientation: string，可选。可选值：landscape、portrait、squarish。
- color: string，可选。可选值：black_and_white、black、white、yellow、orange、red、purple、magenta、green、teal、blue。
- order_by: string，可选。可选值：relevant、latest，默认 relevant。
- page: integer，可选。页码，默认 1。

返回内容：
每张图片返回：
- id
- 描述
- 作者名
- 作者主页
- Unsplash 图片页面
- small/regular 图片 URL
- download_location
- 宽高
## Provider 设计

文件位置设想：
nanobot/providers/unsplash.py

类名：
UnsplashClient

职责：
- 读取 api_key
- 请求 Unsplash 搜索接口
- 处理 HTTP 错误
- 解析 Unsplash 返回结果
- 返回项目内部统一的数据结构

API：
GET https://api.unsplash.com/search/photos

请求头：
Authorization: Client-ID <UNSPLASH_ACCESS_KEY>

请求参数：
- query
- page
- per_page
- order_by
- orientation
- color
- content_filter

返回结构设想：
UnsplashSearchResponse:
- total
- total_pages
- results

UnsplashPhoto:
- id
- description
- alt_description
- author_name
- author_url
- page_url
- small_url
- regular_url
- thumb_url
- download_location
- width
- height
- color
